# Davos Bootstrap – Custom Arch‑Based Linux/GNU Distro

Davos Bootstrap is a lightweight, customizable Arch‑based distribution designed for users who want full control over their system from the very first bootstrapping step. 
This project provides a streamlined bootstrap script that automates the initial setup while keeping the system transparent and user‑driven.

---

## Installation

wget https://github.com/yourusername/davos-bootstrap/archive/refs/heads/main.zip -O davos-bootstrap.zip && unzip davos-bootstrap.zip && cd davos-bootstrap-main && chmod +x bootstrap.sh && ./bootstrap.sh

Alternatively follow these steps to install Davos Bootstrap on your system:

### 1. Download the project  
Download the ZIP archive from this repository.

### 2. Unzip the archive
unzip davos-bootstrap-main.zip

### 3. CD to the project directory

cd davos-bootstrap-main

### 4. Make the Script Executable

chmod +x bootstrap.sh
